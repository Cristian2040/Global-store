### 1. Catalog Management
Stores don't create products from scratch. They search the **Global Catalog** and link existing products to their store.
-   **Pricing**: Stores define their own `retailPrice`.

### 2. Procurement (Restocking)
When stock is low, stores use the "Proveedores" section.
-   **Selection**: View different suppliers offering the same global product.
-   **Order Creation**: Select quantities and create a "Restock Order".
-   **Reception**: Once the supplier delivers, the store verifies it using the **Delivery Code**.
-   **Auto-Update**: Confirming delivery automatically increments the Store's inventory for those products.
